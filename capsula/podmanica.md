# *Capsula podmanica* (Podman)

## Overview
A daemon-less container engine for developing, managing, and running OCI Containers on your Linux System.

## Implementation
Used as the primary container runtime in this workspace (aliased to `docker`).

## Taxonomy
- **Genus**: *Capsula* (Container)
- **Specific Epithet**: *podmanica* (of Podman)
- **Classification**: *Capsula podmanica*
