# *Gestor modelorum* (Model Manager)

## Overview
A suite of tools for managing AI model providers, rotations, and endpoint configurations within the OpenClaw environment.

## Components
- **`tools/rotate-model.sh`**: Shell entry point for switching active models.
- **`tools/rotate-model.js`**: Core logic for updating `config.yaml` with new model aliases.

## Usage
Used to dynamically switch between providers (Flash, Opus, Kimi, Qwen) to optimize for task complexity or cost.

## Taxonomy
- **Genus**: *Gestor* (Manager)
- **Specific Epithet**: *modelorum* (of models)
- **Classification**: *Gestor modelorum*
