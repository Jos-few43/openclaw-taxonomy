# *Monitor usus* (Usage Monitor)

## Overview
A system for tracking token consumption, API calls, and cost metrics across different AI models and sessions.

## Components
- **`tools/usage-tracker.js`**: Real-time logging of model responses and usage metadata.
- **`tools/track-usage.cjs`**: CommonJS module for injecting usage tracking into Node-based tools.

## Usage
Provides the data source for usage visualization and budgeting.
