# *Automator scriptarum* (Script Automator)

## Overview
A framework for triggering and managing the execution of complex CLI scripts and multi-step workflows.

## Components
- **`tools/trigger.sh`**: Universal shell wrapper for executing internal project scripts.
- **`tools/cli-triggers.js`**: JavaScript logic for parsing arguments and routing to the correct tool.

## Usage
Provides a unified interface for the user (or AI agents) to run project-specific commands without knowing the full underlying paths.

## Taxonomy
- **Genus**: *Automator* (Automator)
- **Specific Epithet**: *scriptarum* (of scripts)
- **Classification**: *Automator scriptarum*
