# *Capsula distributa* (Distrobox)

## Overview
A tool to use any Linux distribution inside your terminal, using Podman or Docker to create containers of your favorite distribution.

## Implementation
Used for creating mutable environments on the Bazzite host for specific dev-tooling needs.

## Taxonomy
- **Genus**: *Capsula* (Container)
- **Specific Epithet**: *distributa* (distributed)
- **Classification**: *Capsula distributa*
