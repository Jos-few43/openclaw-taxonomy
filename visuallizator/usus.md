# *Visuallizator usus* (Usage Visualizer)

## Overview
Tools for rendering usage data into human-readable formats, charts, or terminal summaries.

## Components
- **`tools/visualize-usage.cjs`**: Script to process usage logs and output formatted reports.

## Usage
Invoked via the Nexus skill or manual trigger to see current burn rate and distribution.
